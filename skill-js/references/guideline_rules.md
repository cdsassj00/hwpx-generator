# 행안부 AI 친화 행정문서 작성 가이드라인

출처: 행정안전부 보도자료 2026.3.24.

## 핵심 원칙 5가지

### 1. 주어·서술어 명확히
| 금지 (개조식) | 권장 (서술식) |
|---|---|
| AI 시대 문서 체계 혁신 필요 | 정부는 AI 시대에 맞는 문서 체계를 혁신할 필요가 있음. |
| 가이드라인 배포 완료 | 행정안전부는 전 부서에 가이드라인을 배포함. |

### 2. 표준 번호체계 준수
```
Ⅰ. (대제목)
  1. (중제목)
    가. (소제목)
      1) → 가) → (1) → (가) → ① → ㉮
```

### 3. 셀 병합 금지
- **금지**: rowspan/colspan 병합 → `merge_cells()` 절대 사용 금지
- **권장**: 단순 표, 필요시 표 2개로 분리

### 4. 불필요한 서식 제거
- 제목 박스 제거
- 음영·배경색 최소화

### 5. AI 인식 수준별 정보 구조
| 정보 종류 | AI 인식 수준 |
|---|---|
| 일반 문자 | 정확하게 인식 |
| 단순 표 | 명확하게 인식 |
| 문단 간 관계 (특수기호 ㅁ,ㅇ,-) | 인식 오류 발생 |
| 줄·칸 병합 표 | 복잡할수록 오류 증가 |
| 표 안의 표 | 오류 다수 발생 |

## 전/후 예시

### 전 (기존 방식)
```
┌─────────────────────────────────┐
│ AI 시대 행안부 행정문서 가이드라인 보고 │
└─────────────────────────────────┘
서면보고  2026.3.○○.(금)  혁신행정담당관 000

1 추진 배경
○ AI시대 사람과 AI모두 읽고 쓰기 쉬운 문서 필요
```

### 후 (AI 친화 방식)
```
서면 보고 / 2026. 3.○○.(금) / ○○○과장(홍길동)

AI 시대 행안부 행정문서 작성 가이드라인(안) 보고

Ⅰ. 추진배경
  ○ 정부는 사람과 AI 모두 쉽게 읽고 작성할 수 있는
     표준화된 문서 작성 가이드라인을 수립할 필요가 있음.
```

## Claude 내용 변환 체크리스트

생성 전 반드시 확인:
- [ ] 모든 문장에 주어가 있는가?
- [ ] 문장이 ~함, ~임, ~예정임, ~있음으로 끝나는가?
- [ ] 번호체계가 Ⅰ→1→가 순서를 따르는가?
- [ ] `merge_cells()` 를 사용하지 않았는가?
- [ ] 특수기호(ㅁ,ㅇ,-) 단독 사용 없이 BULLET 딕셔너리로 대체했는가?
